This is a simple library management web application built with:

Node.js

Express

MongoDB

JWT authentication

Basic HTML + Bootstrap frontend

The system allows users to register, log in, browse books, and borrow them.


Authentication :

User registration

User login

Password hashing using bcrypt

JWT authentication

Protected routes




Books : 

View all books

Filter by category and page range

Add new books

Borrow books

Return books


Cart : 

Add books to cart

Remove books from cart

Borrow all books in cart

Cart is user-specific
